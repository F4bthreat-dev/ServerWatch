## 📋 Sobre o Projeto

**ServerWatch** é um dashboard web desenvolvido para monitoramento de servidores em tempo real. 
A aplicação fornece métricas essenciais como uso de CPU, memória, disco, rede e processos 
ativos, tudo em uma interface intuitiva e responsiva.

Este projeto foi criado por **f4bthreat-dev (Fabriccio "Threatfull")** como uma ferramenta 
para administradores de sistemas e desenvolvedores que precisam acompanhar o desempenho 
de seus servidores de forma visual e acessível.

---

## 🚀 Funcionalidades

- 📊 **Dashboard em tempo real** com atualização automática
- 💻 **Métricas detalhadas**: CPU, Memória, Disco e Rede
- 🔄 **Listagem de processos** (top 10 por uso de CPU)
- 📈 **Gráficos dinâmicos** para visualização de dados
- 🎨 **Design moderno** e responsivo
- 🔒 **API REST** para integração com outras ferramentas

---

## 🛠️ Tecnologias Utilizadas

### Backend
- **Python 3.8+**
- **Flask 2.3.2** - Framework web
- **psutil 5.9.5** - Coleta de métricas do sistema

### Frontend
- **HTML5, CSS3, JavaScript**
- **Chart.js** - Visualização de dados em gráficos
- **Bootstrap 5** - Design responsivo
